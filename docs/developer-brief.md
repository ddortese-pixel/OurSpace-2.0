# OurSpace 2.0 — Developer Brief

## Project Overview
OurSpace 2.0 is a social networking platform built on Base44. We need a Capacitor specialist to wrap the existing web app into native iOS and Android builds for App Store and Google Play submission.

## Scope of Work
1. Install and configure Capacitor in the Base44 project
2. Configure iOS and Android targets
3. Integrate native plugins (camera, notifications, deep linking)
4. Build signed release binaries for both platforms
5. Submit to Apple App Store and Google Play Store
6. Handle review feedback and resubmission if needed

- Backend: Base44 managed
- Target: iOS 15+ / Android 8+
