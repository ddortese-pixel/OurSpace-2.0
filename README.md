# OurSpace 2.0

> *The Digital Mirror: Your Space, Your Rules.*

A nostalgic yet modern social networking platform built for 2026. We've taken the creative soul of the early web and rebuilt it with elite security, lightning-fast performance, and total user sovereignty.

## 🌟 Core Features

### 🎨 Decorate Your Corner of the Internet
- **The Digital Mirror** — Drag, drop, and skin your profile with widgets, custom CSS-lite themes, and animated backgrounds
- **Profile Soundtracks** — Bring your vibe to life with integrated music players
- **Interactive Guestbooks** — Let friends leave their mark with stickers and voice notes

### 📰 Chronological & Human-Centric
- **The Underground Feed** — No AI-slop. Pure reverse-chronological feed of people you actually follow
- **Verified Human Discovery** — Toggle the Human-Only Filter to see only real-person content
- **Algorithm Reset** — Start fresh while keeping your Inner Circle intact

### ⚡ Next-Gen Features
- **Social Search** — Search audio transcripts in Reels and Live posts
- **Serialized Stories** — Follow creators through "Series" and "Chapters"
- **The Shield (E2EE)** — End-to-End Encrypted conversations

## 🗃️ Entity Schema

| Entity | Description |
|--------|-------------|
| Profile | User profiles with full customization |
| Post | Feed posts (text, image, video, series) |
| Friend | Friend connections & requests |
| Message | E2EE direct messages |
| Comment | Threaded comments with nested replies |
| WallPost | Guestbook-style profile wall posts |
| Notification | In-app activity notifications |
| Story | 24-hour ephemeral stories with polls |
| Reel | Short-form video with audio transcripts |

## 🚀 Tech Stack
- **Frontend:** React (Base44)
- **Backend:** Base44 managed backend
- **Mobile:** Capacitor (iOS + Android — in progress)
- **Auth:** Base44 Auth
- **Storage:** Base44 CDN
